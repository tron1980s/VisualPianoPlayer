# Roblox MIDI Piano

Small Go app for macOS and Windows that converts a `.mid` file into Roblox Virtual Piano key presses.

## Run The GUI

```sh
go run .
```

Choose a MIDI with the native OS picker, adjust playback, then press Start. The start/stop hotkey also works while Roblox is focused.

You can also drag a `.mid` or `.midi` file onto the window. Dropped files are copied into `imported-midi/` and loaded from there.

Use Quick Select to pick any discovered MIDI from the app folder or `imported-midi/`. The gear button opens Song Slots, where keys `1` through `9` can be bound to songs. Slot hotkeys ignore keyboard events generated by playback, so a song pressing `1` through `9` will not switch songs by itself.

## Run Headless

```sh
go run . -cli
```

By default, the app prefers `026-2004-Chopin - Etude Op-10 No-4 (SHYBAY).mid` when it exists.

On macOS, the app needs Accessibility/Input Monitoring permission because it listens for global hotkeys and posts keyboard events while Roblox is focused. If the hotkey does nothing, allow the terminal or the built binary in System Settings, then restart it.

On Windows, the app uses a low-level keyboard hook for global hotkeys and `SendInput` for playback.

## Useful Flags

```sh
go run . -dry-run
go run . -list-map
go run . -transpose 12
go run . -speed 0.8
go run . -mode tap
go run . -disable-sustain
go run . -auto-sustain 900ms
```

`hold` mode is the default because it follows MIDI note-off timing and sounds closer to the original. MIDI sustain pedal events (`CC64`) are simulated by holding notes until the pedal lifts. If a MIDI has no pedal data and sounds dry, use `-auto-sustain 700ms` to `-auto-sustain 1400ms` as a fake pedal tail. Use `tap` mode if Roblox misses held notes or a note gets stuck.

The default playback speed is `1.20x`.

The built-in map includes the requested `y`/`2` swap.

The default macOS hotkey keycode is `41`, which is the physical `ö` key on a German Mac keyboard. The default Windows hotkey keycode is `186`, usually `VK_OEM_1`, the `ö` key on a German Windows layout. If your layout differs, run with `-hotkey-keycode`.
